### Mercado Pago production environment

Configure these variables in **Vercel → Project Settings → Environment Variables** and scope them to **Production**:

| Variable | Usage |
| --- | --- |
| `NEXT_PUBLIC_SITE_URL` | Canonical HTTPS production URL used to build the webhook URL. |
| `NEXT_PUBLIC_MP_PUBLIC_KEY` | Mercado Pago production Public Key exposed to Checkout Bricks. |
| `MP_ACCESS_TOKEN` | Private production Access Token, read only by server routes. |
| `MP_WEBHOOK_SECRET` | Webhook signature secret generated by Mercado Pago. |

Redeploy after adding or changing variables. `NEXT_PUBLIC_MP_PUBLIC_KEY` is embedded during `next build`; private variables remain available only to server functions.
